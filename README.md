Ansible Blackbox Exporter
=========

This role installs zsh with a custom .zshrc and sets it as default

Requirements
------------

- Ansible >= 2.9, may work with other versions, but no guarantee

Example Playbook
----------------

    - hosts: zsh
      roles:
         - { role: kornkalle.ansible_zsh }
